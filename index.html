Glossary

  • Branch: Given End User input, it directs you to another Scene.
  • Playhead: Movable cursor that updates Scene Preview window with specified position in the Scene.
  • Scene: Collection of tracks.
  • Scene Preview: Approximation of what the scene should look like at any given point. The Preview will be interactive.
  • Track: Represents the lifetime and transitions and branching of an object.
  • User Interaction: Input from the End User.
